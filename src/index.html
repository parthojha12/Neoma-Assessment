Real-time Weather App API

You can try entering each route using the routes given below

  • GET /weathers : Get all locations user has added in database collection. Example: http://localhost:8080/weathers
  • POST /weathers/:lat/:lon : User can add weather forecast of any location by latitude(lat) and longitude(lon). Example:http://localhost:8080/weathers/-29.8579/31.0292
  • POST /weathers/:cityName : User can add weather forecast of any location by City (cityName). Example:http://localhost:8080/weathers/delhi
  • GET /weathers/:identifier : User can get weather forecast data of that location from database collection by using ID or City.
  • PUT /weathers/:identifier : User can update weather data of that particular location from database by using its ID or City name.
  • DELETE /weathers/:identifier : Delete weather data from database collection by ID or City.
  • Example for /weathers/:identifiers : http://localhost:8080/weathers/delhi or http://localhost:8080/weathers/65852bacb9ec64a796ad9780
